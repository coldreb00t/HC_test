# HARDCASE - Мобильное приложение для тренировок

## О проекте

HARDCASE - это мобильное приложение для фитнес-тренеров и их клиентов, разработанное с использованием React Native и Expo. Приложение предоставляет удобный интерфейс для управления тренировками, отслеживания прогресса и взаимодействия между тренерами и клиентами.

## Технологии

Приложение разработано с использованием следующих технологий:

- **React Native** - фреймворк для разработки кроссплатформенных мобильных приложений
- **Expo** - платформа для упрощения разработки React Native приложений
- **TypeScript** - типизированный JavaScript для более надежного кода
- **Supabase** - бэкенд-сервис, предоставляющий API для хранения данных, аутентификации и авторизации

## Функциональность

Приложение предоставляет две роли: клиент и тренер.

### Функциональность для клиентов:
- Просмотр запланированных тренировок
- Отслеживание прогресса выполнения тренировок
- Отметка выполненных упражнений и подходов
- Ведение заметок к упражнениям
- Управление профилем

### Функциональность для тренеров:
- Управление списком клиентов
- Создание и редактирование тренировок для клиентов
- Отслеживание прогресса клиентов
- Управление упражнениями и их параметрами

## Установка и запуск

### Предварительные требования
- Node.js (рекомендуется последняя стабильная версия)
- npm или yarn
- Expo CLI (`npm install -g expo-cli`)
- Мобильное устройство с Expo Go или эмулятор

### Установка

1. Клонируйте репозиторий:
```
git clone https://github.com/your-username/hardcase-mobile.git
cd hardcase-mobile
```

2. Установите зависимости:
```
npm install
```
или
```
yarn install
```

3. Настройте переменные окружения в файле `.env`:
```
SUPABASE_URL=your-supabase-url
SUPABASE_ANON_KEY=your-supabase-anon-key
```

4. Запустите приложение:
```
npx expo start
```

5. Отсканируйте QR-код с помощью приложения Expo Go на вашем устройстве или запустите на эмуляторе.

## Структура проекта

```
expo-app/
├── assets/               # Изображения, шрифты и другие статические файлы
├── src/
│   ├── components/       # Переиспользуемые компоненты
│   ├── constants/        # Константы приложения, включая маршруты
│   ├── hooks/            # Пользовательские хуки
│   ├── lib/              # Библиотеки и утилиты (Supabase, API и др.)
│   ├── navigation/       # Навигационная структура приложения
│   └── screens/          # Экраны приложения
│       ├── auth/         # Экраны аутентификации
│       ├── client/       # Экраны для клиентов
│       └── trainer/      # Экраны для тренеров
├── App.tsx               # Корневой компонент приложения
├── app.json              # Конфигурация Expo
├── package.json          # Зависимости и скрипты
└── tsconfig.json         # Конфигурация TypeScript
```
